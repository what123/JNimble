这是一个极简的 JNimble 插件示例，展示了菜单注册和页面路由两个核心能力。

本插件演示了什么

  • 侧边栏菜单：通过 HookRegistry 向 admin.layout.sidebar 钩子注册了一个导航链接，点击即可进入本页面。
  • 页面路由：通过 RouteRegistry 注册了 /index 路由，渲染当前页面。
  • 权限控制：插件声明了 demo-plugin.view 权限，只有被授予该权限的角色才能看到菜单和访问页面。

关键代码

// 注册侧边栏菜单
context.hooks().register("admin.layout.sidebar",
    new HookViewContribution("plugin/demo-plugin/fragment/sidebar", null, 100, "demo-plugin.view", null));

// 注册页面路由
context.routes().register(new RouteDefinition("/index", RouteMethod.GET, "plugin/demo-plugin/page/index", "demo-plugin.view"));
